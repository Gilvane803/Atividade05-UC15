#  Sistema PG-Formaturas — Sistema de Gestão de Formaturas

---
# Tecnologias Utilizadas
Este projeto utiliza apenas tecnologias abordadas no curso:

- **Java**
- **NetBeans IDE**
- **MySQL** (banco de dados relacional)
- **JPA / Hibernate** (persistência)
- **Swing** (interface gráfica desktop)
- **MVC** (padrão de arquitetura)

## Objetivo do Software
O sistema **PG-Formaturas** foi desenvolvido para auxiliar empresas de organização de formaturas a gerenciar:

- Escolas parceiras  
- Turmas e cursos formandos  
- Alunos formandos  

Ele centraliza as informações em um único ambiente, tornando o processo mais rápido, seguro e organizado.

---

## Funcionalidades do Sistema (Requisitos)

###  **Requisitos Funcionais**

- **RF02** — Cadastrar, editar, listar e excluir escolas  
- **RF03** — Cadastrar, editar, listar e excluir turmas vinculadas às escolas  
- **RF04** — Cadastrar, editar, listar e excluir formandos por turma  
- **RF06** — Exibir registros em tabelas (JTable) com pesquisa  
- **RF07** — Exibir mensagens de validação e confirmação ao usuário  

---

### **Requisitos Não Funcionais**
- **RNF01** — Utilizar Java como tecnologia principal  
- **RNF02** — Utilizar MySQL como banco de dados  
- **RNF03** — Interface intuitiva para uso no desktop  
- **RNF05** — Persistência implementada com JPA / Hibernate
